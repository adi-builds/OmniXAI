# 🛠️ Installation

Clone the repository:

```bash
git clone <https://github.com/adi-builds/OmniXAI>
```

Go to client folder:

```bash
cd client
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Run the Project

```bash
npm run dev
```
