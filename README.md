# Omnix AI

This project is being built using modern web technologies with a clean and scalable architecture.

## 🚀 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Lucide React

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
